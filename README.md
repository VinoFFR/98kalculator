# 98kalculator

A modern scientific calculator for Linux (Wayland).

98kalculator is a fast, lightweight, and well-designed calculator built with **Python** and **PyQt6**. It’s made for users who want something more capable and better looking than the default system calculator, without unnecessary complexity.

## Why 98kalculator?

Most Linux calculators are functional but basic.
98kalculator focuses on **speed, usability, and a clean visual style**, while providing the scientific features people actually use.

* **Modern UI** – Dark theme with subtle neon accents and smooth animations
* **Scientific functions** – Trigonometry, logarithms, powers, roots, and factorials
* **Keyboard-friendly** – Full support for keyboard and numpad input
* **Fast startup** – Opens instantly and stays responsive
* **Extras** – Includes a few hidden behaviors for curious users

## Features

* **Basic operations**: `+`, `-`, `×`, `÷`, `%`
* **Scientific functions**: `sin`, `cos`, `tan`, `log`, `ln`, `√`, `x^y`, `1/x`, `x^2`, `x!`
* **Constants**: `π`, `e`
* **Dual display**: Shows both calculation history and current input

## Installation

### Requirements

* Linux (tested on Wayland / Hyprland)
* Python 3

### Install

```bash
git clone https://github.com/VinoFFR/98kalculator.git
cd 98kalculator
python3 install.py
```

This will:

1. Install required dependencies (`PyQt6`, `PyInstaller`)
2. Build a standalone binary
3. Add the application to your system menu

### Uninstall

```bash
python3 uninstall.py
```
